# 云达诺项目 - V1.0.1

本仓库包含云达诺FM33LG0XX测试系统的代码。

## 目录结构

- **云达诺/** - 主项目代码目录
  - `Drivers/` - 驱动程序
  - `Inc/` - 头文件
  - `Src/` - 源代码文件
  - `MDK-ARM/` - Keil MDK工程文件
  - `MF-config/` - 配置文件
  - `README.md` - 项目说明
  - `README_V1.0.1.md` - V1.0.1版本更新说明

## 版本说明

当前版本：**V1.0.1**

查看详细更新内容：[云达诺/README_V1.0.1.md](./云达诺/README_V1.0.1.md)

- ✅ 工位检测扩展至10个工位
- ✅ 新增独立阀门测试流程
- ✅ 优化阀门开关检测逻辑

## 快速开始

1. 克隆仓库
2. 使用Keil MDK打开 `云达诺/MDK-ARM/FM33LG0XX_Tester.uvprojx`
3. 编译并下载到目标设备
